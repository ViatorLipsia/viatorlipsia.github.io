Beschreibung

AppGameKit ist eine leistungsstarke Spielentwicklungs-Engine, ideal für Anfänger, Hobbyisten und Indie-Entwickler. Mit dem BASIC-Scriptsystem von AppGameKit kann jeder schnell Apps und vorallem Spiele für mehrere Plattformen programmieren. Mit nur wenigen Befehlen können Sie Demos / Spiele auf mobilen Geräten ausführen. Bob Duffy, von Developer Relations Intel Corporation, sagte: "AppGameKit ist ein großartiges plattformübergreifendes Entwicklungswerkzeug. Es ermöglicht Entwicklern, Apps schnell zu optimieren und zu kompilieren."

Für Rapid Prototyping können Spiele entwickelt und auf den von Ihnen ausgewählten mobilen Geräten bereitgestellt werden. AppGameKit erledigt die ganze Arbeit für Sie - AppGameKit macht es einfach, Apps für jede Plattform zu programmieren, zu kompilieren und dann zu exportieren. Sie müssen sich nur auf Ihre Spiel- / App-Idee konzentrieren. So einfach ist das!

Plattformen
Sie können Spiele mit AppGameKit auf Windows-, Mac- oder Linux-Geräten erstellen. Die Spiele, die Sie damit erstellen, können anschließend exportiert werden.
  • Windows
  • Mac
  • Linux
  • iOS
  • Android (inkl. Google, Amazon & Ouya)
  • HTML5
  • Raspberry Pi (kostenlos von der TGC-Website)

AppGameKit IDE
  • Plattformübergreifender Geany-Editor
  • IntelliSense
  • Code-Faltung
  • Farbschemata
  • Bearbeiten Sie mehrere Projekte
  • Schneller Kompiler
  • Kompletter Debugger
  • Broadcast direkt zum Gerät
  • Exportieren nach exe, app, ipa, apk



Befehle
